# Обновить прогноз погоды по маршруту

## Intent

**Problem:** Погодный календарь содержал прогноз от 15 июля, включая уже прошедшие дни и устаревшие риски для оставшегося маршрута.

**Approach:** Обновить утро, день и вечер на 17–26 июля по текущим почасовым данным Open-Meteo для фактических точек маршрута, отдельно выделив дождь при въезде в Гродно и неблагоприятные условия на Августовском канале.
