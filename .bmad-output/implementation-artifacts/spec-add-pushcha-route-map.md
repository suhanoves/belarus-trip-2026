# Добавить полную карту велотрека по Беловежской пуще

## Intent

**Problem:** Ссылка Яндекса показывает только восемь промежуточных точек и не позволяет увидеть исходную линию izi.TRAVEL вместе со всеми 33 объектами.

**Approach:** Встроить в день 17 июля интерактивную карту исходного трека с полной линией, пронумерованными объектами, переходами к рассказам и геолокацией, а для офлайн-приложений дать GPX с той же линией и точками. Явно оставить izi.TRAVEL и указатели парка основным ориентиром на развилках, поскольку исходная геометрия не является пошаговой навигацией.
